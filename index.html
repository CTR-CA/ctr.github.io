The following items are the responsibility of the Installing Company or Dealer:

  1. Ensure that a qualified technician performs the installation
  2. Installation of the unit, battery, fuel tank, and fuel lines as specified in the applicable Carrier Transicold installation drawings. Instalation drawings are supplied with every unit; they may also be obtained directly from Carrier Transicold.
  3. Install all options specified by the customer. These may include air chutes, return air bullkheads, light bars, etc. Installation instuction should be provided with each of these items by the manufacturer.
  4. Add a minimum of 15 gallons of diesel fuel to the fuel tank of the refrigeration unit. Use #2 diesel fuel if the ambient temeprature is expected to remain above +10℉ (-12℃); if the ambient temeprature is equipped with an EES.

The following items are the responsibility of the Technician, Installing Company and/or Dealer performing the Pre-Delivery Inspection and unit delivery to owner;

  1. Fill out identification data (above and below) and inspect the installation of unit and all options. Verify that the installation has been performed to the requierments specified in the Carrier Transicold installation drawings.
  2. Perform the Pre-Delivery Inspection, ensuring the use of the appropiate form for the model unit. EES units must be documented with CARB
  3. As the owner takes delivery of the unit (from Delaer or installer) AND the unit is put into service, the in-service date must be stamped on the unit serial plate, as this.in-service data verifies the beginning data of ownership and warranty.
  4. Within ten (10) days of the in-service date, the unit must be registred with the Carrier Transicold Warranty Department.
  5. For completing steps 3 and 4 above, Dealers should refer to the Warranty Administration Policies & Procedures Manual, P/N 62-10068 (Dealer Operating Manual, Section 6,1), for specific instruction. Trailer and truck body OEM's should contact the selling Carrier Transicold Dealer for copies of these detailed instructions.

Caution: Take all necessary steps to ensure that the unit will not start while technician is working on any moving parts of the unit. Unit can start automaticlly. Disconnect battery cable when appropiate.

  1. Pre-Start Unit Check/Installation
Torque unit mounting bolts and nuts (60 fr-lbs).
Check fuel tank and fuel line installation (both supply and return lines should be one inch from bottom of fuel tank).
Check fuel and oil line connections for leaks.
Check wire connections and grounds for tightness.
Check all wire harnesses, cables, hoses and fuel lines for clearance of chafing.
Check battery vable connections and routing.
Check engine coolant hose connections for leaks.
Check engine oil level. Add______qts.
Check engine coolant level. Add______qts.
Open all refrigeration system service valves.
Install latest software from TransCentral.
Record software version installed______.
  1. Set up of APX*
Check & Set Control system Configurations (See page3)

Check & Set Data Recorder (See page3)
  1. Initial Start-Up of Engine
Install refrigeration service gauges
Start unit automatically. Warning buzzer should sound for five seconds prior to automatic unit start-up. If the buzzer does not sound, repair before proceeding.
Enter 11℉ (-12℃) Set Point, Continuous Run operation.
Listen for unusual noises, check all lines, houses and gasket areas for leaks.
Check condenser and evaporator for proper airflow.
After 5 minutes of operation, record:
  • Discharge Pressure @ gauges ______.
  • Suction Pressure @ gauges ______.
  • Ambient Temp. @ micro ______.
  • Return Air Temp. @ micro ______.
Check refrigerent level Add______lbs.
Check compressor oil level.
  1. Engine Break-In
Check fuel level in tank, add if required.
Open trailer doors. Start unit and enter a -22℉(-30℃) Set Point, Continuous Run operation.(In extremly cold weather, enter an 80℉(27℃) Set Point) Operate unit for 4-6 hours minimum.
